Convert any sidebar into a responsive, icon-activated sidebar for mobile users, works with any sidebar, works with woocommerce filters on sidebar.

== Description ==

Transform your wordpress sidebar in a mobile responsibe sidebar, mobile users can access your sidebar widgets on the go just clicking the sidebar button.

Your sidebars will slide onscreen, works great with woocommerce filters! 

This mobile friendly plugin will bring next level usability to your wordpress website.

= How it Works =

* install
* sidebars are now hidden by default on small screens only
* mobile users can now see a round button on the bottom of the scree to open/close sidebars

= Features =

* Responsive sidebar open/close button;
* Elementor Compatible;
* Works with any theme;
* Great for woocommerce filters;
* Lightweight, this plugins uses only wordpress core scripts;

== Installation ==

= Plugins Directory =

1. Install `Mobile Responsive Sidebar`
1. Activate `Mobile Responsive Sidebar` from Plugins page

= Direct Dowload =

1. Unzip the downloaded zip file.
1. Upload the plugin folder into the `wp-content/plugins/` directory of your WordPress site.
1. Activate `Mobile Responsive Sidebar` from Plugins page
